# LovePets

Site estático de apresentação da loja LovePets, criado para servir como um projeto de agenda/desenvolvimento de sistemas.

## Descrição

Esta página apresenta uma loja de produtos para pets com visual leve, limpo e acolhedor. O layout usa o framework **W3.CSS** para deixar o HTML simples, sem estilos embutidos, e mantém a navegação fixa no topo.

## Estrutura

- `index.html` — página principal do site
- `img/` — pasta com as imagens dos produtos

## Funcionalidades

- Navegação fixa no topo usando `nav`
- Menu com título à esquerda e links alinhados à direita
- Seção de produtos com três cards lado a lado
- Blocos de conteúdo com leitura confortável e espaçamento agradável
- Paleta de cores suave, transmitindo sensação de paz e alegria

## Tecnologias

- HTML5
- W3.CSS (via CDN)
